Attribute ของ tag ต่อไปนี้มีอะไรบ้าง และใช้งานอย่างไร

Tag Attribute and How to use it
<h>

ใช้กำหนดหัวข้อตามความสำคัญ มีตั้งแต่ h1 - h6

<p>

align เป็นแอททริบิวต์ภายใต้แท็ก <p> เป็นการกำหนดการจัดวางตำแหน่ง ของข้อความ หรือตัวอักษรในตำแหน่ง ต่าง ๆ ของหน้าเว็บเพจ หรือช่องตาราง ดังนี้

title ใช้กำหนดไตเติลให้บทความ จะแสดงผลเมื่อนำเมาส์ไปชี้

<div>

id, class ใช้แบ่งชนิดของ div

title กำหนดไตเติล

style กำหนดสไตล์แบบ inline

align กำหนดตำแหน่ง

<img>

alt text ระบุข้อความเพิ่มเติมสำหรับภาพ

border ระบุความกว้างของเส้นขอบรอบภาพ

height ระบุความสูงของภาพ

hspace ระบุช่องว่างด้านซ้ายและด้านขวาของภาพ

ismap ระบุภาพเป็นฝั่งเซิร์ฟเวอร์ภาพแมป

longdesc ระบุ URL ไปยังรายละเอียดของภาพ

src ระบุ URL ของภาพ

usemap ระบุภาพเป็นฝั่งไคลเอ็นต์ภาพแมป

vspace ระบุช่องว่างด้านบนและด้านล่างของภาพ

width ระบุความกว้างของภาพ

<a>

href ใช้ระบุลิงค์ปลายทาง

<ul>

style กำหนดสไตล์แบบ inline

<ol>

start ใช้เริ่มกำหนดอันดับเริ่มแรกของลิสต์

reversed เรียงลำดับถอยหลัง

type ใช้กำหนด type ของการนับ เช่น 1 A a I i

<table>

style กำหนดสไตล์แบบ inline

<form>

accept-charset กำหนด charset สำหรับที่ใช้ในการกรอกฟอร์ม

action กำหนดลิงค์สำหรับการส่งเดต้าจากฟอร์ม

autocomplete กำนหนด auto complete ให้เป็น เปิด/ปิด

enctype กำหนดการเข้ารหัสของ http method post

method กำหนด http method

name ตั้งชื่อให้ฟอร์ม

novalidate กำหนดให้ฟอร์มไม่สามารถ validated เมื่อทำการซับมิตข้อมูล

target ใช้กำหนดการแสดงผลหลังจากทำการซับมิตข้อมูล

<button>

type ใช้กำหนด type สำหรับปุ่ม

name ใช้กำหนดชื่อสำหรับปุ่ม
